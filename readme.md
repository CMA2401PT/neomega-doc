## neomega-doc
### 介绍
本仓库包含了 **neomega** 的文档。lumega文档涵盖了安装说明、lua 插件编写、coromega api等内容。

### 本地开发
依赖环境：
- Node.js v18.16.0+
- 包管理器 yarn
```shell
npm install --global yarn
yarn install
```
运行：
```shell
yarn docs:dev
```
构建：
```shell
yarn docs:build
```
